{% comment %} Override of django-unfold's `unfold/helpers/app_list.html` that adds support for nested sidebar items. Unfold's Python layer (`UnfoldAdminSite._get_navigation_items`) already recursively processes child `items` arrays — applying `has_permission`, `active`, `link_callback`, and `badge` to each level. Only the upstream template renders flat (one level), so a custom template is enough to unlock subtree navigation. We render each item via the recursive `unfold/helpers/app_list_item.html` partial; an item that carries its own `items` list becomes a collapsible subtree (Alpine.js drives the expand/collapse exactly like top-level collapsible groups do).
